Detailed Description

A classe CutBox inativa um objeto retangular determinado pelo usuário.

Com uma classe concreta, o usuário vai escrever "cutbox" no arquivo para inativar um objeto retangular entre as posições x0 e x1, y0 e y1, z0 e z1.
