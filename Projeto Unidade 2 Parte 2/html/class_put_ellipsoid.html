Detailed Description

A classe PutEllipsoid cria uma elipsoide determinada pelo usuário.

Com uma classe concreta, o usuário vai escrever "putellipsoid" no arquivo para criar uma elipsoide com centro x, y e z, raios rx, ry e rz, nas cores r, g, b e a.
