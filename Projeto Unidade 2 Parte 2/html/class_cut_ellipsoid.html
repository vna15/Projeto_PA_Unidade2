Detailed Description

A classe CutEllipsoid inativa uma elipsoide determinada pelo usuário.

Com uma classe concreta, o usuário vai escrever "cutellipsoid" no arquivo para inativar uma elipsoide com centro x, y, z e raios rx, ry, rz.
